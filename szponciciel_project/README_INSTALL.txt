
==============================================
Instrukcja: jak zbudować EXE przez GitHub Actions
==============================================
1) Rozpakuj ZIP.
2) Na GitHub utwórz repo (Public, nazwa: szponciciel).
3) Kliknij: Upload files → przeciągnij CAŁĄ zawartość tego folderu (ważne: .github/workflows/main.yml).
4) Commit changes.
5) Wejdź w Actions → Build Electron App → Run workflow.
6) Po zakończeniu (zielony znacznik) pobierz z sekcji Artifacts: Szponciciel-installer.
7) W środku będzie instalator „Szponciciel Setup 1.0.1.exe”.
8) Zainstaluj i uruchom. Dane trzymane są w Dokumenty/Szponciciel.
9) Ustawienia (zębatka): zmiana koloru akcentu.
10) Import plików: przycisk „Importuj pliki” albo przeciągnij na dropzone.
